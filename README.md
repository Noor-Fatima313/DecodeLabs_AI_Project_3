# DecodeLabs_AI_Project_3
## AI Recommendation System

## Project Overview

This project implements a simple recommendation system using Python. The program accepts user interests, compares them with predefined categories using pattern matching, and recommends relevant items.

## Features

- Accepts multiple interests
- Uses dictionary-based recommendation logic
- Displays recommended items
- Calculates a similarity score
- Beginner-friendly implementation

## Technologies

- Python 3
- Lists
- Dictionaries
- Functions
- Loops
- Conditional Statements

## How to Run

1. Open the notebook in Jupyter Notebook or Google Colab.
2. Run each cell from top to bottom.
3. Enter your interests when prompted.
4. View the recommended items and similarity score.
